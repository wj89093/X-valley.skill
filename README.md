# X-Valley 新道蓝谷招商助手

金帝·新道蓝谷生命科学园 AI 招商话术助手。

## 数据源

| 数据 | 说明 |
|------|------|
| 话术库 | 园区基础信息、FAQ、拒绝应对话术 |
| 最新动态 | 入驻企业动态、园区新闻公告 |

两个数据源均托管于飞书文档，由官方实时维护。

## 使用方法

将此 skill 安装到支持 OpenClaw skill 协议的平台，skill 将自动读取飞书文档获取话术和动态数据。

## 安装要求

- 支持 OpenClaw skill 协议
- 支持 `feishu_fetch_doc` 工具调用
