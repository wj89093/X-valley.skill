# X-Valley 新道蓝谷招商助手

金帝·新道蓝谷生命科学园 AI 招商话术助手。

## 介绍

面向生物医药产业园区招商场景的 AI Agent skill。帮助招商人员解答企业咨询、匹配政策、留资跟进。

## 适用场景

- 企业来电/来访咨询园区基本情况
- 介绍 OPC 社区（1人起租共享实验室）
- 解答 GMP 认证、环评排污等合规问题
- 匹配萧山 5213 等政策补贴
- 应对「比价格」「看其他园区」等拒绝场景
- 预约参观、留资跟进

## 使用方法

将此 skill 安装到支持 OpenClaw skill 协议的平台，参照平台文档配置即可使用。

## 数据源

- 基础话术：内置于 SKILL.md
- 最新动态：飞书文档（见 SKILL.md 内置链接）
- 政策数据：萧山区 5213 计划等（见 SKILL.md 内置政策条款）

## 注意事项

- 房源价格、具体租金等请以招商经理确认为准
- 政策补贴以官方文件为准，本 skill 仅供参考
- 如需更新动态数据，请联系招商负责人
